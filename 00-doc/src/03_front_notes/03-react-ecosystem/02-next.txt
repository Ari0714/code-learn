--- core feature

Next.js 之所以「火」，主要因为它是一个现代、全能、高性能的 React 应用框架，解决了开发 React 应用中最痛的几个问题，比如：
首屏加载慢、SEO 难做、配置复杂、部署麻烦

为 React 应用提供更好的开发体验 + 更强的部署能力 + 更全面的性能支持


1. 支持 SSR + SSG + ISR
2. 零配置 + React 整合好
3. 优秀的 SEO 支持
4. 内置路由系统
5. 可部署到任意平台(vercel, docker...)



----------------------------------------------------------concept----------------------------------------------------------------------------------------   

css是描述性语言，原生
less是css预处理语言，在css基础拓展，最终被编译成css




css 核心特点：
1. 支持@$变量名
2. 支持嵌套
3. + - * / 支持运算符
4. 可写复用样式，类似函数
5. 可读性高。适用大型项目



--- 示例
- 变量
#div1{
	color: @maincolor
}


- 嵌套语法
nav {
  ul {
    li {
      color: red;
    }
  }
}


- 运算
.margin {
  margin: @base * 2;
}
